# 💻 코다리 — 현재의 기술적 요구사항(FE/BE, 결제창 등)을 무시하고, '감성적인 방안'과 '작동하는 오브젝트/애니메이션'에 초점을 맞추어 MVP 범위를 재정의하는 개발 계획을 작성해 주세요. 핵심은 RoomCanvas 내에서 오브젝트 배치(Grid Snapping)와 애니메이션(Framer Motion)을 중심으로 기능을 묶어 MVP의 핵심 사용자 경험을 완성하는 것입니다.

### 🚀 RoomCanvas MVP 개발 계획 (감성적 UX 중심)  
**목표:** 2개월 내 RoomCanvas 핵심 사용자 경험 완성 (결제/백엔드 무시)  

---

#### 🔍 **1. 핵심 기능 정의 (MVP 범위)**
**1.1 Grid Snapping 시스템**  
- **기능:**  
  - 아이소메트릭 그리드 생성 (대각선 격자)  
  - 오브젝트 드래그 시 자동 스냅 ( snapping tolerance: 10px)  
  - 스냅 효과: 애니메이션 (Framer Motion)으로 **반짝임 + 색상 변화**  
- **감성적 요소:**  
  - 그리드 라인: **부드러운 테두리 + 반투명 효과**  
  - 스냅 성공 시 **3D 회전 애니메이션** (1초간)  

**1.2 오브젝트 애니메이션 (Framer Motion 기반)**  
- **기능:**  
  - 오브젝트 드래그 시 **움직임 애니메이션** (bounce, drag trail)  
  - 스냅 후 **입력 효과**: 색상 변화 + 투명도 조절  
  - 오브젝트 삭제 시 **fade-out + 회전 애니메이션**  
- **감성적 요소:**  
  - 오브젝트 색상: **하이픈 디자인 (soft gradients)**  
  - 애니메이션 속도: **부드러운 0.3s~0.5s**  

**1.3 감성적 UI/UX 디자인**  
- **기능:**  
  - 전체 화면: **아이보리 배경 + 블랙 그리드 라인**  
  - 오브젝트 라이브러리: **카드형 아이콘 (hover 시 3D 효과)**  
- **감성적 요소:**  
  - 텍스트 폰트: **Sans-serif + 약간의 shadow 효과**  
  - 투명도: **오브젝트 80%, 배경 100%**  

---

#### 📅 **2. 개발 일정 (2개월 기준)**  
**Phase 1: 기초 구조 (Week 1-2)**  
- Grid Snapping 시스템 구현  
- Framer Motion 기본 애니메이션 적용  
- 색상/폰트/배경 감성 디자인 완성  

**Phase 2: 상호작용 확장 (Week 3-4)**  
- 오브젝트 라이브러리 추가 (5개 이상의 기본 오브젝트)  
- 스냅/드래그/삭제 애니메이션 완성  
- UI/UX 감성 디자인 최종 조정  

**Phase 3: 테스트 & 개선 (Week 5-6)**  
- 사용자 테스트 (애니메이션 반응성, 스냅 정확도)  
- 성능 최적화 (Framer Motion 애니메이션 최적화)  
- 감성적 요소 피드백 반영  

**Phase 4: MVP 완성 (Week 7-8)**  
- 모든 기능 통합 (Grid + 애니메이션 + 감성 디자인)  
- **"Showcase" 탭 추가** (사용자에게 애니메이션 예시 보여주기)  
- 최종 테스트 및 문서화  

---

#### 🎨 **3. 감성적 디자인 예시**  
- **오브젝트 색상:**  
  - 기본: 푸른색 (soft blue gradient)  
  - 활성화 시: 주황색 (orange glow)  
- **애니메이션 예시:**  
  - 드래그 시: **파란색 트레일**  
  - 스냅 시: **3D 회전 + 투명도 80% → 100%**  
- **UI 요소:**  
  - 버튼: **둥근 모서리 + hover 시 색상 변화**  
  - 텍스트: **폰트 크기 16px, shadow 1px**  

---

#### ✅ **4. MVP 성공 기준**  
- **기능:**  
  - Grid Snapping 100% 정확도  
  - Framer Motion 애니메이션 100% 작동  
  - 오브젝트 라이브러리 5개 이상 제공  
- **감성적 UX:**  
  - 사용자 피드백: "직관적", "재미있는", "부드러운"  
  - 애니메이션 속도: **0.3s~0.5s (최적화)**  
  - 색상/디자인: **사용자에게 감성적 인상을 주기**  

---

#### 🛠️ **5. 테스트 계획**  
- **내부 테스트 (Week 5):**  
  - Grid Snapping 정확도 확인 (50회 이상 테스트)  
  - 애니메이션 반응성 확인 (다양한 기기에서 테스트)  
- **외부 테스트 (Week 7):**  
  - 10명 사용자에게 MVP 시연  
  - 피드백 수집 (감성적 UX, 애니메이션, 스냅 경험)  

---

**결론:**  
MVP는 **"작동하는 오브젝트"와 "부드러운 애니메이션"**에 집중하며, 감성적 디자인을 통해 사용자에게 **재미와 직관적인 경험**을 제공할 것입니다. 기술적 요소는 후순위로, **사용자 경험(UX)**을 최우선으로 삼는 전략을 실행합니다.
